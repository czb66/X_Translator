# 安装指南

## 快速开始

### 1. 准备图标文件

扩展程序需要图标文件才能正常安装。你需要创建以下三个图标文件：

- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)
- `icon128.png` (128x128 像素)

**临时解决方案**：你可以：
1. 使用在线工具生成简单图标（如 https://www.favicon-generator.org/）
2. 创建纯色占位图片
3. 从免费图标库下载翻译相关的图标

### 2. 加载扩展程序

#### Chrome/Edge 浏览器：

1. 打开浏览器，访问 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `X_Translator` 文件夹
5. 扩展程序安装完成！

#### Firefox 浏览器：

1. 访问 `about:debugging`
2. 点击"此 Firefox"
3. 点击"临时载入附加组件"
4. 选择 `manifest.json` 文件

### 3. 使用扩展程序

1. 访问 X (Twitter) 网站（https://twitter.com 或 https://x.com）
2. **在输入框中**：选择文字后会自动翻译成推文语言（或默认英语）
3. **在普通页面**：选择文字后会翻译成中文
4. 点击"替换"按钮将译文替换原文（仅在输入框中可用）
5. 点击"复制"按钮复制翻译结果

## 功能说明

### 自动识别场景

- **写字模式**：在输入框（推文输入框、回复框等）中选择文字
- **看书模式**：在普通页面内容中选择文字

### 智能目标语言

- **写字时**：自动检测当前推文的语言，翻译成推文语言
- **看书时**：始终翻译成中文

### 一键替换

- 仅在"写字模式"下显示"替换"按钮
- 点击后直接将原文替换为译文
- 支持各种类型的输入框

## 故障排除

### 翻译不工作？

- 检查网络连接
- 确认已访问 X (Twitter) 网站
- 查看浏览器控制台是否有错误信息

### 替换功能不工作？

- 确保在输入框中选择文字
- 某些特殊输入框可能需要手动复制粘贴
- 检查翻译是否成功（翻译失败时无法替换）

### 图标显示错误？

- 确保三个图标文件都存在
- 检查图标文件格式（PNG）
- 检查图标文件尺寸是否正确

## 注意事项

- 需要网络连接以使用翻译服务
- 使用 Google Translate 免费 API，可能有使用限制
- 翻译质量取决于 Google Translate
- 某些特殊输入框可能不支持自动替换，需要手动复制粘贴
